# The Number Guessing Game (Cmpsc 132 Final Project)

## Description

This is a number guessing game built using the Godot Engine with the P4Godot (Python for Godot) plugin.
The player tries to guess a randomly generated number, with feedback provided through a graphical user interface that expands on the terminal version.

---

## Requirements

* Godot Engine (version: 4.6 stable)
  *(Must match this version to run properly)*

---

## How to Run

1. Download and extract the ZIP file
2. Open Godot Engine
3. Click **Import**
4. Select the `project.godot` file from the extracted folder
5. Click **Import & Edit**
6. Press **F6** to run the game

---

## Plugin Setup (Important)

If the project does not run:

1. Go to **Project → Project Settings → Plugins**
2. Ensure **P4Godot** is enabled

---

## How to Play

* Click on start and choose difficulty
* Enter a number into the input field
* Submit your guess
* The game will tell you if your guess is too high or too low
* Continue guessing until you find the correct number or run out of guesses

---

## Troubleshooting

**Project won’t open**

* Make sure you are using the correct Godot version

**Game does not run**

* Ensure the P4Godot plugin is enabled

**Errors appear**

* Re-extract the ZIP file and try again

---
